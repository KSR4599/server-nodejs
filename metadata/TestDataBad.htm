HAPI Server for the TestDataBad dataset

This server supports the HAPI __VERSION__ API specification for delivery of time series data.

The parameters in this dataset are code-generated and intended for testing HAPI clients and verifiers. All parameter types and output formats are supported and N-D array parameters are served. The dataset contains one dataset; the data associated with this dataset is the same as that in the catalog TestData but has many common data and metadata errors.

The metadata for the scalarcats parameter in dataset0 has a non-HAPI standard field of x_categorymap that will probably change or be removed in the future. dataset0 also has a x_maxDurations field that indicates the maximum duration of the request above which a HAPI 1408 error will be given (too much time or duration requested).

The source code for this server is available in a GitHub Repository.

The server responds to GET requests to the following HAPI endpoints:

  • capabilities - list the API version and output options
  • catalog - list the datasets that are available
  • info - list the parameters in the dataset; example:
    • info?id=dataset0
  • data - stream data content for a parameter; examples:
    • data?id=dataset0&parameters=scalar&time.min=2000-01-01&time.max=2000-01-02
    • data?id=dataset0&parameters=vector&time.min=2000-01-01&time.max=2000-01-02
    • data?id=dataset0&parameters=spectra&time.min=2000-01-02&time.max=2000-01-02
    • data?id=dataset0&parameters=scalar,vector&time.min=2000-01-02&time.max=2000-01-02
    • data?id=dataset0&time.min=2000-01-02&time.max=2000-01-02
