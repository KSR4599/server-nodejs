HAPI Server for TestData datasets

This server servers software-generated parameters that can be used to test a HAPI client (see the HAPI project page for a list of existing clients).

The parameters include most types that are supported by the HAPI specification along with variations that are expected to be found in real datasets (non-uniform cadence, fill values, empty time intervals).

Note that if the time interval 1970-01-01T00:00:10.000 to 1970-01-01T00:00:20.000 is requested for any parameter, the body of the response for a HAPI CSV or HAPI Binary request (with no header) is empty, e.g. this request.

This server supports the HAPI 2.0 API specification for delivery of time series data.

The server responds to GET requests to the following HAPI endpoints:

  • capabilities - list the API version and output options
  • catalog - list the datasets that are available ( total)
  • info - list information about parameters in a dataset, e.g,:
  • data - stream data for parameters in a dataset. Note that this server supports the non-HAPI input of attach=false, which tells the server to not send data as an attachment; typically a browser will display the data in a window instead of prompting a download (e.g., this request).
