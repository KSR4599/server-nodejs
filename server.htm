HAPI Server for __CATALOG__ data

This server supports the HAPI __VERSION__ API specification for delivery of time series data.

The parameters in this dataset are code-generated and intended for testing HAPI clients. The parameters include most types of allowed HAPI parameters including scalars paramters of integer, double, string and isotime are and array parameters of type integer and double (but not yet string or isotime). The HAPI output formats of CSV and binary are supported (but not yet JSON). Experimental support for alternative output formats fcsv and fbin are also included and will be eventually be removed. The metadata for the scalarcats parameter has a non-HAPI standard field of categorymap that will probably change or be removed in the future.

The source code for this server is available in a GitHub Repository.

The server responds to GET requests to the following HAPI endpoints:

  • capabilities - list the API version and output options
  • catalog - list the datasets that are available
  • info - list the parameters in the dataset; example:
    • info?id=TestData
  • data - stream data content for a parameter; examples:
    • data?id=TestData&parameters=scalar&time.min=2000-01-01&time.max=2000-01-02
    • data?id=TestData&parameters=vector&time.min=2000-01-01&time.max=2000-01-02
    • data?id=TestData&parameters=spectra&time.min=2000-01-02&time.max=2000-01-02
    • data?id=TestData&parameters=scalar,vector&time.min=2000-01-02&time.max=2000-01-02
    • data?id=TestData&time.min=2000-01-02&time.max=2000-01-02
