HAPI Server for __CATALOG__ data

This server supports the HAPI __VERSION__ API specification for delivery of time series data. The server consists of the following endpoints that respond to HTTP GET requests:

  • capabilities - list the API version and output options
  • catalog - list the datasets that are available
  • info - list the parameters in the dataset; example:
    • info?id=TestData
  • data - stream data content for a parameter; examples:
    • data?id=TestData&parameters=scalar&time.min=2000-01-01&time.max=2000-01-02
    • data?id=TestData&parameters=vector&time.min=2000-01-01&time.max=2000-01-02
    • data?id=TestData&parameters=spectra&time.min=2000-01-02&time.max=2000-01-02
    • data?id=TestData&parameters=scalar,vector&time.min=2000-01-02&time.max=2000-01-02
    • data?id=TestData&time.min=2000-01-02&time.max=2000-01-02
